# 邮箱功能的开发
### 发送邮件

smtp类，主要使用 fsockopen 访问smtp服务器，直接发送邮件

功能：系统管理员，可以给所有用户发送邮件

	循环，即可实现群发！
	
	实现站内信：只需要简单修改即可，用户需要提供自己的邮箱账号和密码，才能实现！
		解析邮箱，拼凑服务器名--smtpserver，获取用户名--smtpuser等

### 邮箱注册验证
第一步：邮箱验证，必须发送邮件

	发送 HTML 类型的邮件，自己拼凑出网址链接，即用户接收用户激活注册的页面，发送激活邮件给用户。
	目的：接收用户发送的激活码，验证身份，实现激活。

第二步：制造激活码，防止用户伪造：使用MD5加密，加密的字符串，可以自己任意选择，只有用户不知道你的算法，即不能伪造
	可行方案，用户名+注册时间即可。

第三步：数据库存储基本信息，首先用户的注册信息，用户以后登录需要验证身份，激活码信息，等用户点击链接，
	再次查询数据库，进行验证，确认用户身份，验证成功后，修改数据库，即实现用户的注册了。
	
第四步：结束了，用户可以登录了。
